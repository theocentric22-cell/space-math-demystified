# Space Math Demystified

Welcome to **Space Math Demystified**! This is an open-source web platform dedicated to breaking down advanced astrophysics, orbital mechanics, and cosmic phenomena using senior secondary school mathematics.

## 🚀 Project Mission
Many students find space science intimidating because of the complex math. This project aims to prove that if you understand basic algebra, geometry, and sequences, you can understand how the universe works.

## 📬 Current Articles
* **Weighing a Star with High School Algebra:** Discovering how algebraic cancellation allows us to calculate the mass of giant celestial bodies.

## 🛠️ Built With
* HTML5 & CSS3 (Custom Deep-Space Theme)
* MathJax (For rendering LaTeX equations)
* GitHub Pages (Hosting)
