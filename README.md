# 🌌 Space Math Demystified

An interactive web platform dedicated to breaking down advanced astrophysics, orbital mechanics, and cosmic phenomena using senior secondary school mathematics. 

This project aims to prove that if you understand basic algebra, geometry, and sequences, you can understand how the universe works.

---

## 🚀 Active Modules

* **The Ultimate Diet Plan: Weighing a Star with High School Algebra** Discovering how algebraic cancellation allows us to calculate the mass of giant stars using planetary velocity and distance.

* **Slicing Up Space: How a Cone Explains the Orbit of Mars** Exploring how conic section geometry (ellipses, parabolas, and hyperbolas) dictates the movement of planets and comets.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Thetawise-inspired Dark Dashboard)
* **Math Rendering:** MathJax Engine for beautiful LaTeX equations
* **Hosting:** GitHub Pages
